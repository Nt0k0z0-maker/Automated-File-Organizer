Automated File Organizer (Python)
Overview
The Automated File Organizer is a Python automation script designed to automatically sort and organize files within a selected directory based on file type.
The script scans a folder, detects file extensions, creates categorized folders if they do not exist, and moves files into their appropriate locations. This reduces manual file management and improves productivity.
Features
Automatically scans a directory
Detects file extensions
Creates folders dynamically
Organizes files into categories:
Documents
Images
Videos
Music
Archives
Others
Reduces clutter in directories
Lightweight and efficient automation
Technologies Used
Python 3
os module (file system operations)
shutil module (file movement)
